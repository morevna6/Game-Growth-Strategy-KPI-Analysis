# Game Growth Strategy & KPI Analysis

## Project Overview
This project focuses on identifying growth opportunities by analyzing key performance indicators (KPIs) across different games.

---

## Analysis Focus

### 1. Key Metrics
- Revenue  
- Total users  
- Paying users  
- Ad spend  

### 2. KPI Analysis
- Conversion rate (Paying Users / Total Users)  
- ARPU (Revenue per user)  
- CPU (Cost per user)  
- ROI / Efficiency  

### 3. Growth Strategy Insights
- Which games have the highest growth potential  
- Where conversion can be improved  
- Which metrics should be optimized first
